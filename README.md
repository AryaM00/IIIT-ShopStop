# IIIT ShopStop

## How to Run

### Frontend
1. Navigate to the `frontend` directory:
   ```bash
   cd frontend
2. Install dependencies
    ```bash
    npm install
3. Start the app
    ```bash
    npm run dev
#

1. Navigate to the `backend` directory:
   ```bash
   cd backend
2. Install dependencies
    ```bash
    npm install
